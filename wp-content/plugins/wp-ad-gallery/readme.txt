== Description ==

WP AD Gallery jQuery plugin for Wordpress. Easy add gallery slideshow without code. Add shortcode using WP AD Gallery shortcode button on post editor.

Now, you can add different options for each posts using shortcode. WP AD Gallery also support for Custom Post Type.

You can see an example of the code in action here:

http://coffeescripter.com/code/ad-gallery/

USE

Insert shortcode by click WP AD Gallery button on post editor and follow the options 

== Installation ==

1. You can either use the automatic plugin installer or your FTP program to upload it to your wp-content/plugins directory the top-level folder. Don't just upload all the php files and put them in /wp-content/plugins/.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Insert shortcode by click WP AD Gallery button on post editor and follow the options
5. That's it!!
